Welcome!

If you want to learn about some of my projects, you’ve come to the right place. While not an exhaustive list, here are some of my projects that I’m particularly proud of.

A bit about me: I’m Gabe, a high school senior looking to make quality programs, particularly games.

Books

  • Hello Scratch! (Manning, 2017) - a book I wrote alongside my sister and mother. In it, you will learn how to create retro-style games in Scratch, including both programming and creating the art for the games. I wrote the programming sections.

Games

  • city-guesser - A companion game to GeoGuessr where you are given a city name, and must guess the country based off it. Playable here. Written in Python.
  • Untitled Puzzle Game - a game that I am working on for my Capstone project. It is a puzzle game made in Unity that involves a grid, looking into the future, and boxes. A playable demo should be available in Spring 2023, with a full release likely later that year.

Scratch Games

Note: Some of these projects may have some slight bugs due to the Scratch 3.0 transition.

  • PartiBlu - My (current) magnum opus. It was featured on Scratch and was on the front page for a looooong time. (It was also included as an example for remixing in a Chinese translation of another Scratch book, for some reason.)
  • x^2 - I consider it the sequel to PartiBlu, but it’s somewhat different (and a lot more random).
  • Grey Runner - one of my first “serious” games.
  • Collaborative Clicker - was front page curated. I was quite proud of the button animation, which I made in Blender.
  • Bonus - a game I made that was intended to be played on mobile devices, but when I made it, Scratch did not work on mobile! Perhaps I was a bit ahead of the curve on that one.
  • Retromakers - the projects that I made for Hello Scratch!
